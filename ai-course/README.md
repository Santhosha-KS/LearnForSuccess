# 🤖 Complete AI Field Guide for Software Engineers

An interactive, self-paced AI course built for senior engineers — covering AI/ML fundamentals, LLMs, RAG, Prompt Engineering, and practical AI stacks.

**Built with:** React 19 + Vite &nbsp;|&nbsp; **Author:** Santhosha 😊

---

## 🚀 Quick Setup (2 minutes)

### Prerequisites
Make sure you have these installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- npm (comes with Node.js)

Check your versions:
```bash
node -v
npm -v
```

### Steps

```bash
# 1. Clone the repo
git clone https://github.com/Santhosha-KS/LearnForSuccess.git

# 2. Go into the project folder
cd LearnForSuccess/ai-course

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```

Then open your browser and go to:
```
http://localhost:5173
```

That's it — the course opens instantly! 🎉
